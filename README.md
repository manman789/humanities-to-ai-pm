# 人文背景转 AI 产品经理 · Skill 工具包

帮中文、语言学、教育、内容、传播类背景的同学，
用真实经历申请 AI 产品经理岗位。

**核心原则：不撒谎，不吹牛，把真实背景翻译成产品语言。**

---

## 怎么用

### 方式一：配合 AI 直接使用
把 `SKILL.md` 上传给 Claude / ChatGPT，
再把 `references/` 和 `assets/` 里的文件一起上传，
然后告诉 AI 你的情况，它会按照工具包的逻辑帮你。

### 方式二：做成 GPTs 分享给别人
1. 打开 chatgpt.com → 探索 GPTs → 创建
2. 把 `SKILL.md` 内容粘贴到「指令」里
3. 把其他所有 `.md` 文件上传到「知识」里
4. 发布后得到链接，分享给需要的人

---

## 先从这里开始

打开 `SKILL.md`，找到「第一步：先判断你在哪个阶段」。

对应你的情况，按指引走就行。

## 文件说明

| 文件 | 用途 |
|------|------|
| `SKILL.md` | 总入口，先看这个 |
| `references/background-mapping.md` | 把你的专业背景翻译成 AI PM 方向 |
| `references/portfolio-projects.md` | 没有项目时，从这里选一个做 |
| `references/resume-bullet-bank.md` | 把经历改写成产品语言的句子 |
| `references/jd-analysis-rubric.md` | 拆解一个 JD，找匹配点和缺口 |
| `references/interview-bank.md` | 练习面试追问，不是给你背答案 |
| `assets/resume-template.md` | 简历整体结构 |
| `assets/prd-template.md` | 写 AI 产品 PRD |
| `assets/portfolio-case-template.md` | 把项目写成展示文档 |
