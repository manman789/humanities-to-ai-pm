# Portfolio Projects

Use these project ideas for humanities-to-AI-PM candidates. Prefer projects that can be explained with user problem, product design, AI workflow, and evaluation plan.

## Recommended Project Types

### AI Writing Feedback Assistant

Target users:

- Students, teachers, exam candidates, workplace writers.

Core problem:

- Users need feedback beyond grammar: structure, logic, style, audience fit, and revision suggestions.

AI capabilities:

- Prompted evaluation, rubric-based scoring, rewrite suggestions, examples, iterative feedback.

Portfolio evidence:

- PRD, prototype, prompt design, evaluation rubric, sample before/after outputs.

### Teacher Lesson Planning Copilot

Target users:

- Teachers preparing lesson plans, exercises,课堂活动, and differentiated materials.

Core problem:

- Lesson preparation is repetitive and must fit learning goals, student level, and classroom time.

AI capabilities:

- Generate lesson outline, exercises, examples, questions, and differentiated tasks.

Portfolio evidence:

- User journey, PRD, prompt workflow, quality review checklist, teacher-in-the-loop process.

### Chinese Learning Companion

Target users:

- Chinese learners, international students, children, adult learners.

Core problem:

- Learners need contextual practice, correction, and confidence-building conversation.

AI capabilities:

- Dialogue practice, error correction, explanation, spaced review, personalized topics.

Portfolio evidence:

- Conversation flow, error taxonomy, feedback design, safety and age-appropriate design.

### Knowledge Base Q&A Product

Target users:

- Students, employees, researchers, customer service teams.

Core problem:

- Users cannot quickly find accurate answers in long documents.

AI capabilities:

- RAG, citations, source snippets, query clarification, answer confidence.

Portfolio evidence:

- Information architecture, retrieval flow, answer evaluation, hallucination mitigation.

### AIGC Content Operations Tool

Target users:

- Content operations, marketers, editors, course designers.

Core problem:

- Content production needs speed, consistency, review, and channel adaptation.

AI capabilities:

- Topic expansion, style adaptation, batch rewriting, compliance review, publishing workflow.

Portfolio evidence:

- Workflow map, content standards, prompt templates, review states, metrics.

## Portfolio Case Structure

1. Project title and one-sentence value proposition.
2. Target users and scenario.
3. User pain points and evidence assumptions.
4. Product goal and success metrics.
5. MVP scope.
6. AI workflow.
7. Key interaction flow.
8. Evaluation plan.
9. Risks and mitigations.
10. Iteration roadmap.
